# 🗺️ Velotask 开发路线图 (Roadmap)

本文档概述了 Velotask 的未来开发计划。我们的目标是将 Velotask 打造成最高效、最令人愉悦的任务管理应用。

## 🚀 核心体验

专注于完善任务管理应用的标准功能，补齐当前缺失的基础能力。

- [ ] **本地通知 (Local Notifications)**:
  - 临近截止日期 (DDL) 的提醒功能。
  - 每日特定时间的任务摘要通知。
- [ ] **重复任务 (Recurring Tasks)**:
  - 支持按天、周、月以及自定义间隔重复任务。
- [ ] **子任务 (Subtasks)**:
  - 允许将主任务拆分为更小的、可勾选的子项。
- [ ] **搜索功能 (Search)**:
  - 全局搜索栏，支持按标题或描述查找任务。

## 🎨 组织视图

增强用户查看和组织数据的方式。

- [ ] **日历视图 (Calendar View)**:
  - 月/周日历组件，直观展示带有日期的任务。
- [ ] **统计仪表盘 (Statistics Dashboard)**:
  - 展示任务完成趋势的图表。
  - 基于重要性和完成速度的“效率评分”。
- [x] **标签系统 (Tags/Labels)**:
  - 支持自定义颜色标签，扩展现有的 DDL/TDL/WTD 分类。
- [ ] **拖拽排序 (Drag & Drop)**:
  - 允许在列表中手动拖拽调整任务顺序。

## ☁️ 云端备份

突破单机限制，提供多端同步体验。

- [ ] **云同步与备份 (Cloud Sync)**:
  - 用户认证系统 (注册/登录)。
  - 多设备数据同步。
- [ ] **数据导入/导出**:
  - 支持导出数据为 CSV/JSON 格式。

## 🎮 游戏化与高级功能

让生产力变得有趣且智能。

- [ ] **游戏化 (Gamification)**:
  - 连胜统计 (Streaks)：连续完成任务天数。
  - 成就/徽章系统。
- [ ] **自然语言输入 (NLP)**:
  - 自动解析如“明天下午5点买牛奶”的文本为任务。

## 🌍 国际化 (i18n)

- [x] 提取硬编码字符串到 ARB 文件。
- [x] 支持应用内切换 英文 / 简体中文。

---

*本路线图可能会根据用户反馈和贡献情况进行调整。*
