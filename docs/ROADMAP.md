# 🗺️ Velotask 开发路线图

本文档概述了 Velotask 的未来开发计划。

我们的目标是将 Velotask 打造成最高效、最令人愉悦的任务管理应用。

短时间不会变化的已完成功能用**粗体**标记。
最近正在处理和改进的功能计划用 *斜体* 标记。
需要征求意见的计划以 `？` 结尾，详见各 ISSUES。

##  ✨ 基础功能
- [x] **基础任务**：单次任务的基础增删改查。
- [ ] *子任务*：将任务拆分为更小的、可勾选的子项？
- [ ] 重复任务：支持按天、周、月及自定义间隔的重复任务。

## 🚀 增强功能
- [x] **自定义标签**：自定义标签，扩展现有分类。
- [x] **甘特图视图**：展示任务的甘特图。
- [x] **自然语言输入**：自动解析如“明天下午5点买牛奶”的文本为任务。
- [x] *仪表盘视图*：展示任务完成趋势、指标的图表。
- [x] *紧急度指标*: 设计一个优先级引擎，为任务设定紧急度指标？
- [ ] *通知提醒*：提醒临近截止的任务以及每日摘要通知。
- [ ] *优先级提升*：对于达到阈值的紧急度的任务提升优先级。
- [ ] *数据导入/导出*：支持导出数据为 CSV/JSON 格式。
- [ ] 搜索功能：按标题或描述进行全局搜索。
- [ ] 日历视图： 月/周日历组件，直观展示带日期的任务。
- [ ] 任务排序：按优先级排序，DDL排序，以及手动拖拽调整序。

## 🎨 UI 与主题呈现
- [x] **图标设计**：设计一个应用图标。
- [ ] *统一设计语言*：设计一个强调速度感的UI，全局统一使用这个设计。
- [ ] 主题隔离管理：主题与组件分离管理。
- [ ] 配色设计：可选择的配色设计。
- [ ] 平台适配：在不同平台上为了适配体验，做不同的设计。

## 🗄️ 其他
- [x] **Drift 数据库**：迁移到 Drift 数据库。
- [x] **多语言支持**：应用内切换简体中文和英文。
- [ ] 云同步与备份：多设备数据同步？
- [ ] 游戏化：成就/徽章系统。

---

*本路线图可能会根据用户反馈和贡献情况进行调整。*
