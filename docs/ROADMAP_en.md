# 🗺️ Velotask Development Roadmap

This document outlines the future development plan for Velotask.

Our goal is to make Velotask the most efficient and delightful task management application.

Features that are complete and unlikely to change in the short term are marked in **bold**.  
Features that are currently being worked on or improved are marked in *italics*.  
Plans that require feedback are marked with `?` — see individual issues for details.

## ✨ Basic Features
- [x] **Basic tasks**: Basic CRUD for one-time tasks.
- [ ] *Subtasks*: Break tasks into smaller, checkable sub-items?
- [ ] Recurring tasks: Support for recurring tasks by day, week, month, and custom intervals.

## 🚀 Enhanced Features
- [x] **Custom tags**: Custom tags to extend existing categories.
- [x] **Gantt chart view**: Display tasks in a Gantt chart.
- [x] **Natural language input**: Automatically parse text like "buy milk tomorrow at 5 PM" into a task.
- [x] *Dashboard view*: Charts showing task completion trends and metrics.
- [x] *Urgency indicator*: Design a priority engine to assign urgency indicators to tasks?
- [ ] *Notification reminders*: Reminders for upcoming deadlines and daily summary notifications.
- [ ] *Priority escalation*: Escalate priority for tasks whose urgency reaches a certain threshold.
- [ ] *Data import/export*: Support exporting data to CSV/JSON format.
- [ ] Search functionality: Global search by title or description.
- [ ] Calendar view: Monthly/weekly calendar component to visualise tasks with due dates.
- [ ] Task sorting: Sort by priority, due date, and manual drag-and-drop ordering.

## 🎨 UI & Theming
- [x] **Icon design**: Design an app icon.
- [ ] *Unified design language*: Design a UI that emphasises a sense of speed, and apply this design consistently across the app.
- [ ] Theming isolation: Separate theme management from components.
- [ ] Colour schemes: Selectable colour schemes.
- [ ] Platform adaptation: Adapt designs for different platforms to ensure an optimal experience.

## 🗄️ Other
- [x] **Drift database**: Migrate to Drift database.
- [x] **Multi-language support**: Switch between Simplified Chinese and English within the app.
- [ ] Cloud sync & backup: Sync data across multiple devices?
- [ ] Gamification: Achievements / badge system.

---

*This roadmap is subject to change based on user feedback and contributions.*
